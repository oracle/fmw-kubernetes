Install Guide

Install the WebLogic Kubernetes Operator to prepare and deploy Oracle WebCenter Content domain.

    Understand the system requirements and limitations for deploying and running Oracle WebCenter Content with the WebLogic Kubernetes Operator, including the Oracle WebCenter Content cluster sizing recommendations.

    Prepare your environment

    Prepare for creating Oracle WebCenter Content domain, including required secrets creation, persistent volume and volume claim creation, database creation, and database schema creation.

    Create Oracle WebCenter Content domain

    Create Oracle WebCenter Content domain home on an existing PV or PVC and create the domain resource YAML file for deploying the generated Oracle WebCenter Content domain.
